## 🚀 Featured Projects

### 💰 FinanceApp — Personal Finance REST API
> Java 21 · Spring Boot · PostgreSQL · JWT Authentication

A full-featured personal finance management API built from scratch. Features full CRUD for transactions and categories, **user-scoped authorization** (each user sees only their own data), JWT-based authentication, and a `/dashboard` summary endpoint aggregating income, expenses, and balance.

🔗 [`ramiresrodrigodev/financeapp`](https://github.com/ramiresrodrigodev/financeapp)

**Highlights:**
- Secure multi-user architecture with Spring Security + JWT
- Clean layered architecture: Controller → Service → Repository
- RESTful design with proper HTTP semantics
- PostgreSQL with JPA/Hibernate

### 🏗️ Construction Materials Manager
> Java CLI · React (no-build) · localStorage · CSV Export

A full-stack management app for construction material inventory. Java handles backend logic via CLI, while a self-contained React frontend (zero build step required) delivers a polished UI with SVG charts, CSV export, and an AI Assistant tab.

**Highlights:**
- No-build React frontend — just open the HTML
- Real-time SVG charts for inventory visualization
- CSV export for reporting
- Integrated AI Assistant tab
